# Persona: The Docs Author

**Name**: Yuki
**Role**: Technical writer / documentation lead
**Experience**: 4+ years
**Pronouns**: they/them

### Project Context

Yuki is responsible for creating and maintaining documentation across the parallel-powers framework. They need consistent conventions, templates, and tools to produce high-quality docs efficiently.

### Goals
- Create clear, consistent documentation following Diátaxis
- Maintain the glossary and terminology
- Ensure diagrams render correctly across platforms
- Enable self-service documentation for other team members

### Pain Points
- No established documentation conventions
- Inconsistent frontmatter across files
- Diagrams not rendering properly in different viewers
- Hard to maintain cross-references between docs

### How We Help
- [File structure conventions](../12-metadata/file-structure.md) for frontmatter and structure
- [Diagram guidelines](../12-metadata/diagrams.md) for ASCII and Mermaid standards
- [Glossary](../10-glossary/) for terminology management
- [How-to guides](../03-howto/) for documentation templates
