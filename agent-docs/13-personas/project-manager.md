# Persona: The Project Manager

**Name**: Morgan
**Role**: Project manager
**Experience**: 7+ years
**Pronouns**: they/them

### Project Context

Morgan is responsible for planning and tracking work across the parallel-powers framework. They need visibility into milestones, issue status, time tracking data, and the decision history to report to stakeholders.

### Goals
- Track project progress against milestones
- Understand team velocity (human vs. AI-assisted)
- Review design decisions and their rationale
- Generate status reports for stakeholders

### Pain Points
- Outdated roadmaps with no clear milestone tracking
- No visibility into AI-assisted vs. human effort
- Missing release timelines and decision history
- Hard to find the current state of plans and specs
