# Persona: The Operator

**Name**: Jordan
**Role**: DevOps engineer
**Experience**: 3+ years
**Pronouns**: they/them

### Project Context

Jordan is responsible for maintaining the infrastructure that supports the parallel-powers workflow — GitHub repositories, issue labels, MCP server configurations, and any CI/CD pipelines.

### Goals
- Set up GitHub issue labels for a new project
- Configure MCP servers for time tracking integration
- Maintain consistent project metadata across repositories
- Automate agent documentation validation

### Pain Points
- Missing deployment or setup documentation
- Unclear MCP configuration requirements
- No runbooks for common operational tasks
- Inconsistent label configurations across projects
