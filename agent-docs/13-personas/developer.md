# Persona: The Developer

**Name**: Wei
**Role**: Full-stack developer
**Experience**: 5+ years
**Pronouns**: they/them

### Project Context

Wei is building on top of the parallel-powers framework, either to extend the framework itself or to use the bootstrap kit to initialize a new project. They need clear conventions, working examples, and reference docs.

### Goals
- Initialize a new project quickly using the bootstrap kit
- Understand the agent-docs file structure and frontmatter schema
- Write new documentation that follows established conventions
- Debug issues with frontmatter, diagrams, or broken links

### Pain Points
- Poor documentation of the bootstrap process
- Missing examples of how to create custom personas
- Unclear frontmatter schema or validation rules
- Inconsistent diagram conventions across docs
