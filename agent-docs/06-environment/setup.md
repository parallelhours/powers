# Environment Setup

## Prerequisites

- Python 3.x
- Claude Code (recommended for agent workflows)
- Git

## Setup

No virtual environment or dependencies are required currently. The project is a documentation and conventions framework.

```bash
# Clone the repository
git clone <repo-url>
cd parallel-powers

# (Optional) Install Claude Code
npm install -g @anthropic-ai/claude-code
```
