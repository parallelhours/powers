# Getting Started with parallel-powers

This tutorial walks through initializing a new project with the parallel-powers documentation framework.

## Prerequisites

- An existing git repository
- Claude Code or another AI agent

## Steps

### Step 1: Copy the Bootstrap Kit

Copy the `agent-docs-bootstrap/` directory into your project as `agent-docs/`.

### Step 2: Identify Your Project

Determine your project name and primary purpose.

### Step 3: Create Project Metadata

Create `agent-docs/12-metadata/project.yml` with your project details.

### Step 4: Create CLAUDE.md

Create or update `CLAUDE.md` in the project root with project context and key conventions.

### Step 5: Create Personas

Customize the personas in `agent-docs/13-personas/` for your project.

### Step 6: Validate

Check frontmatter, diagrams, and links.
