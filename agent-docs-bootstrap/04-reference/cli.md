# CLI Reference

Complete reference for command-line interface.

## Commands

### `command-name`

Brief description of what the command does.

**Usage**:
```bash
command-name [options] [arguments]
```

**Options**:

| Flag | Description | Default |
|------|-------------|---------|
| `-h, --help` | Show help message | - |
| `-v, --version` | Show version | - |
| `-c, --config <path>` | Config file path | `./config.json` |

**Examples**:

```bash
# Basic usage
command-name input.txt

# With options
command-name --verbose input.txt output.txt
```

Description of subcommand.
