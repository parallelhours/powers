# Runbooks

Operational procedures and checklists for running the system.

## Contents

- [Deployment](deployment.md) — Deploy procedures
- [Incident Response](incident-response.md) — Incident handling
- [Maintenance](maintenance.md) — Regular maintenance tasks

## When to Use Runbooks

- Execute deployments
- Respond to incidents
- Perform routine maintenance
- Onboard operations team

## Style Guidelines

- Numbered steps in exact order
- Include rollback procedures
- Specify who should execute each step
- Include expected time for each step
