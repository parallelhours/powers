## Getting More Help

If your issue isn't covered here:

1. Check existing [GitHub Issues](https://github.com/user/repo/issues)
2. Search the [Discussions](https://github.com/user/repo/discussions)
3. File a new issue with the `bug` template

## Diagnostic Commands

```bash
# Check version
npm run --version

# Check configuration
npm run doctor

# View logs
npm run logs:recent
```
